# Resistor Colour Code Calculator
A functional tool that can be used to calculate the resistance of a resistor.
Has a clean user interface with dropdown menus (or clickable color swatches) for selecting 3 or 4 color bands.
Use of JavaScript : The screen must instantly update to display the correct resistance value (in ohms) and tolerance percentage whenever the user changes a color.
Can handle edge cases clearly (e.g., displaying an error or placeholder if not all bands are selected yet).
